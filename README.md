### IOT feeder app for AUL STIC

### Hardware

- 1 Led
- 1 resistor
- 1 ultrasonic sensor
- 1 NodeMCU ESP8266
- 5v power supply
- 1 Water level sensor

#### Development

- git clone https://github.com/Tekipeps/IOT-animal-feeder.git

- cd IOT-animal-feeder

- yarn install

- yarn start
